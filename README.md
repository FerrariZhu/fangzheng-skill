# 方正.skill

> 蒸馏自朱方正的产品履历、访谈产品观与决策故事，供 AI Agent 以「方正」的框架做产品决策参谋。

---

## 这是什么

这是一个 Cursor Agent Skill（项目级），把方正在途虎、SHEIN、携程、CTW 等经历中可复用的产品打法，以及本人补充的原则、取舍与关键故事，蒸馏成结构化知识，用于回答产品决策类问题。

**触发场景举例：**

- 「问问方正，这个付费会员页该怎么改？」
- 「方正会怎么看这个营销工具要不要做成平台？」
- 「用方正的框架分析一下 xx系统 要不要 MCP 化」
- 「开卡转化卡住了，方正怎么拆？」
- 「时间很紧又要上会员，体验和进度怎么取舍？」

---

## 方正是谁

**朱方正**，约 9 年产品经验。南京财经大学。

擅长计划和执行的业务产品：目标拆解、迭代逼近结果；在意结果与数据。别人找参谋时期望：**解决方案 + 上线进度**。

代表经历：途虎付费会员与组合售卖工具、SHEIN CLUB 频道转化、Trip 美食林 SEO、CTW 平台 Admin MCP 化。

## Skill 包含的内容

| 类别 | 内容 |
|------|------|
| 个人观 | 说明书、不变原则、冲突取舍、明确反对 |
| 决策框架 | 转化优先、人群分叉、工具化复用、主流程增强、基建杠杆、Agent/MCP、AI 全链路交付 |
| 关键故事 | 途虎超级会员（进度 vs 体验）；SHEIN 弹窗（AB 后再组件化） |
| 案例库 | CTW / 携程 / SHEIN / 途虎等履历案例（`cases.md`） |
| 说话风格 | 先给方案再讲理由；偏温和不硬刚；评审三问原话 |
| 参谋流程 | 澄清 → 先方案 → 原则/框架/反对项 → 案例类比 → 取舍与进度 |

## 如何使用

本仓库已将 Skill 放在项目目录：

```text
.cursor/skills/fangzheng/
├── SKILL.md    # 主入口
└── cases.md    # 案例与关键故事（按需读取）
```

在 Cursor 中打开本项目后，对助手说「问问方正」或「方正会怎么看」即可触发。

无需额外安装；若复制到其他机器，保持上述相对路径即可。
